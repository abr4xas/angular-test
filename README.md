## Prueba básica de AngularJS usando las guías de estilo de John Papa con JSONPlaceholder

### Requerimientos

* Node
* Bower
* Gulp

```bash
$ bower install
``` 

luego: 

```
$ npm install
``` 

Y por ultimo:


```
$ npm start
```
